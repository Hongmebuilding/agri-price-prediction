📚 Machine Learning & Deep Learning
이 프로젝트는 시계열 데이터를 기반으로 농산물 가격을 예측하기 위해 머신러닝 및 통계 기반 기법을 활용합니다. 현재 사용된 알고리즘은 다음과 같으며, 실험 및 비교를 위한 다양한 예측 모델을 학습 및 적용하였습니다:

Linear Regression / Ridge / Lasso: 기본 회귀모델들을 통해 베이스라인 성능을 측정

Random Forest Regressor: 비선형성과 변수 중요도를 고려한 앙상블 회귀

XGBoost / LightGBM: 성능이 우수한 Gradient Boosting 기반 알고리즘 사용

Prophet (optional): 시계열 데이터에 특화된 모델 적용 실험 가능

Rolling mean smoothing: 단순 이동평균을 통해 추세 시각화
